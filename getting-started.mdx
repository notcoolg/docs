Welcome to Azura, your intelligent AI assistant built for everything from quick questions to deep, multi-turn conversations. This guide walks you through signing in, sending your first message, and getting comfortable with the interface.

## What is Azura?

Azura is an AI-powered chat assistant that can:

- Answer questions and explain complex topics
- Help you brainstorm ideas and work through problems
- Write, review, and debug code
- Create interactive demos, documents, and visualizations through the **Artifacts** system
- Remember important details about you across conversations using **Long-term Memory**
- Analyze images and attached files (with Azura 1-Plus)

## Signing In

Azura uses Economic ID authentication. To get started:

1. Navigate to the Azura web app
2. If you aren't already signed in, you will be redirected to [id.ecowestern.net](http://id.ecowestern.net) to authenticate
3. Complete the sign-in (or signup) flow using your email and password
4. Once authenticated, you'll land on the main chat interface

Your conversations, memories, and artifacts are all tied to your account and sync across devices.

## The Interface at a Glance

When you open Azura, you'll see:

| Area | Description |
| --- | --- |
| **Sidebar** (left) | Your conversation history and navigation |
| **Chat Area** (center) | The main conversation feed |
| **Omnibox** (bottom) | The input bar for typing messages and running commands |
| **Artifact Panel** (right) | Slides open when Azura creates interactive content |
| **Header** (top) | Access to settings, theme toggle, and account controls |

## Sending Your First Message

1. Click the **Omnibox** at the bottom of the screen
2. Type your message
3. Press **Enter** or click the **Send** button (the arrow icon)

That's it, Azura will begin responding immediately, streaming text as it thinks.

> **Tip:** Press **Shift \+ Enter** to add a new line in your message without sending it.

## Starting a New Conversation

Click the **New Chat** button in the sidebar (or the `+` icon in the header) to start a fresh conversation. Previous conversations remain in your sidebar and can be revisited at any time.

## Suggested Prompts

When you open a new conversation, Azura shows a few starter suggestions on the welcome screen:

- **Explain a concept** — ask Azura to break down anything confusing
- **Help me brainstorm** — get ideas, outlines, or creative prompts
- **Write some code** — generate, review, or debug code in any language
- **Answer questions** — get accurate, conversational answers on any topic

Click any card to jump right in, or type your own message.
