The **Omnibox** is Azura's input bar, more than just a text field. It supports slash commands, quick actions, model switching, and auto-expanding multiline input, all from a single control at the bottom of the screen.

## Typing a Message

Click the Omnibox and start typing. When you're ready:

- Press **Enter** to send
- Press **Shift \+ Enter** to insert a line break without sending

The Omnibox grows automatically as you type longer messages, up to a maximum height before it scrolls.

---

## The `+` Menu

Click the **`+`** button on the left side of the Omnibox to open the quick-action menu. It has two sections:

### AI Model Selector

Quickly switch between models without typing a command:

- **Azura 1-Basic** — Fast & efficient
- **Azura 1-Consider** — Enhanced reasoning
- **Azura 1-Plus** — Premium performance \+ Vision

### Quick Actions

| Action | Description |
| --- | --- |
| **Attach File** | Upload a document or file to include in your message |
| **Code Snippet** | Insert a code block |
| **Voice Input** | Use your microphone to speak your message |
| **Add Document** | Attach a document for Azura to read and reference |

## Slash Commands

Type `/` in the Omnibox to trigger the command autocomplete. All commands are case-insensitive.

### Model Commands

| Command | What it does |
| --- | --- |
| `/basic` | Switch to Azura 1-Basic (fast & efficient) |
| `/consider` | Switch to Azura 1-Consider (enhanced reasoning) |
| `/plus` | Switch to Azura 1-Plus (premium \+ vision) |

### Conversation Commands

| Command | What it does |
| --- | --- |
| `/clear` | Clears the current conversation and starts fresh |
| `/summarize` | Asks Azura to summarize the conversation so far |
| `/tokens` | Shows current token usage for the conversation |

### Interface Commands

| Command | What it does |
| --- | --- |
| `/settings` | Opens the Settings modal |
| `/theme` | Toggles between Dark and Light mode |
| `/artifacts` | Toggles the Artifact side panel open or closed |
| `/help` | Shows available commands and shortcuts |

### Memory Commands

| Command | What it does |
| --- | --- |
| `/forget` | Clears Azura's long-term memory for your account |

---

## Using Commands

There are two ways to run a slash command:

**Option 1 — Type and autocomplete:**

1. Start typing `/` in the Omnibox
2. A command picker will appear above the input
3. Use the keyboard or mouse to select a command
4. Press Enter to run it

**Option 2 — Type in full:**

1. Type the full command (e.g., `/clear`) in the Omnibox
2. Press Enter

## Keyboard Shortcuts

| Shortcut | Action |
| --- | --- |
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `/` (at start of message) | Open command picker |

## Animated Placeholders

When the Omnibox is empty, it cycles through a set of playful placeholder messages every 15–20 seconds. These are just for fun, you can type your message at any time.

Examples you might see:

- _"Message Azura..."_
- _"The Omnibox: your portal to infinite possibilities"_
- _"I'm all ears... well, all circuits"_
- _"Type away, I don't byte!"_

---

## Tips

- **Accidentally cleared your conversation?** Azura stores all your conversations in the sidebar, you can still navigate to older ones.
- **Want to switch models quickly?** `/basic`, `/consider`, and `/plus` are the fastest way.
- **Writing a long prompt?** The Omnibox supports multi-line input — just use Shift \+ Enter to break lines.
