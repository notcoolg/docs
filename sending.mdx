# Sending Messages

eChat makes it easy to send messages to anyone on the platform, whether it's a quick DM, a group chat, or a channel post. Here's everything you need to know.

## Starting a new conversation

### Direct message (DM)

1. From the **Chat** screen, tap the **compose / new chat** icon (pencil or \+ button).
2. Search for a contact by name or Chattercode.
3. Select the person to open a new conversation.
4. Type your message and hit **Send**.

### From your contacts

1. Go to the **Home** screen to see your contacts list.
2. Tap a contact's name to open a direct conversation.

<Note>
  You can only message people who are in your contacts, or who are part of a shared group or channel.
</Note>

## The message input area

At the bottom of any conversation, you'll find the message input bar. Here's what you can do from there:

| Control | What it does |
| --- | --- |
| **Text field** | Type your message here |
| **Attachment icon** | Open the attachment picker for media and files |
| **GIF icon** | Search and send a GIF |
| **Microphone icon** | Hold to record a voice note |
| **Send button** | Send your message |

## Sending a message

1. Tap the text field at the bottom of the conversation.
2. Type your message.
3. Tap the **Send** button (arrow icon) or press **Enter** on a keyboard.

## Text formatting

eChat supports **Markdown formatting** in messages. Use these shortcuts while typing:

| Format | Syntax | Example output |
| --- | --- | --- |
| **Bold** | `**text**` | **bold text** |
| _Italic_ | `*text*` | _italic text_ |
| `Code` | ``text`` | `inline code` |
| Code block | ````text```` | Formatted code block |
| Bullet list | `- item` | • list item |
| Line break | Shift \+ Enter | New line within a message |

## Editing a sent message

Made a typo? You can edit a message after sending it:

1. **Long-press** (mobile) or **right-click** (desktop) the message.
2. Select **Edit** from the context menu.
3. Make your changes in the input field.
4. Press **Save** or hit Enter to confirm.

Edited messages display a small **(edited)** label so the recipient knows it was updated.

## Deleting a message

1. **Long-press** (mobile) or **right-click** (desktop) the message.
2. Select **Delete** from the context menu.
3. Confirm the deletion.

Deleted messages are removed from the conversation for everyone.

## Read receipts

When you send a message in a DM, you'll see small profile pictures appear beneath your message as recipients read it. This tells you exactly who has seen your message.

In groups and channels, a summary count or avatar stack shows who has read each message.

## Typing indicators

When another person in a conversation is composing a reply, you'll see a **"\[Name\] is typing…"** indicator in the chat. This updates in real time so you know when to expect a response.

## Searching conversations

Use the **search icon** or filter tabs at the top of the chat list to quickly find a specific conversation. You can filter by:

- **All** — every conversation
- **Channels** — only channel conversations
- **DMs** — only direct messages

---

<Tip>
  Press **Shift \+ Enter** in the text box to add a new line to your message without sending it — great for longer messages with multiple paragraphs.
</Tip>
